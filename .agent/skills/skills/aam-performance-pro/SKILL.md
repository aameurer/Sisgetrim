# Skill: Performance & Otimização
Sempre que o usuário pedir uma revisão:
1. Identifique redundâncias no JavaScript (ex: seletores de DOM repetidos).
2. Sugira o uso de `defer` ou `async` para scripts.
3. Verifique se o CSS do Tailwind pode ser otimizado ou se há classes conflitantes.
4. Proponha o uso de `Event Delegation` em listas longas.
